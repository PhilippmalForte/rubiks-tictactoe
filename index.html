Rubik's Tic-Tac-Toe

Dieses Spiel kombiniert Tic-Tac-Toe mit einem Rubik's Cube. Es ist das erste Computerspiel, das ich selbst entwickelt habe. 🙂

Spielregeln
  1. Zeichen setzen: Klicke auf ein leeres Feld.
  2. Ebene drehen: Ziehe danach über den Würfel.
  3. Gewinnen: 3 gleiche Zeichen in einer Reihe auf einer Seite.
Tipp: Ziehe über den Hintergrund, um den Würfel von allen Seiten anzusehen.
